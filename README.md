小程序api接口实例代码

- 简单获取信息
  - 获取用户信息
  - 获取组的信息
- 简单上传数据
  - 上传用户信息
  - 上传组的信息
- 表单上传数据
  - 上传用户信息
  - 上传组的信息
- 个人定制API

- 人脸数据集操作
  - 人脸数据入库
  - 人脸数据搜索
